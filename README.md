## Project Name:
AI-Based Real-Time Object Detection and Distance Monitoring System

## Description:
This project uses YOLO object detection with a laptop camera and an ultrasonic sensor connected to Raspberry Pi.

## Features:
- Real-time object detection
- Distance measurement
- Too Near warning
- Out of Range detection
- Flask live streaming

## Technologies Used:
- Python
- YOLO
- OpenCV
- Flask
- Raspberry Pi
- Ultrasonic Sensor
